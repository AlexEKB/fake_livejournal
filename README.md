# Fake medium.com

#### Описание

Web-приложение, клон medium.com

#### Ruby and Rails version
```
ruby 2.5.1
rails 5.2.2
```
***
### Первый запуск
```
gem install bundler
bundle install
```
Затем подготавливаем базу данных:
```
rails db:create
rails db:migrate
```
Запуск сервера:
```rails s```

### Тесты
```
rspec
```
